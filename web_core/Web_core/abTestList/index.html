A/B测试
A/B 测试是一种“先验”的实验体系，属于预测型结论，与“后验”的归纳型结论差别巨大。A/B测试的目的在于通过科学的实验设计、采样样本代表性、 流量分割与小流量测试等方式来获得具有代表性的实验结论，并确信该结论在推广到全部流量可信。
